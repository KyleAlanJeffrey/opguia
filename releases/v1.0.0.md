# v1.0.0

Initial stable release.

- Full rewrite with simplified data type resolution and inline comments
- Tree-table view with compact 26px rows and search filtering
- Inline values, types, and status dots in the browse tree
- Write support for writable variables
- Collapsible detail dialog with full node attributes
- Auto-scan for local OPC UA servers on common ports
- Native desktop window via pywebview
- pip installable (`pip install opguia`)
